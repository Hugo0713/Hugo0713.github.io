计网-物理层

发表于 2025-02-24 更新于 2025-02-27 分类于 Networking 阅读次数：
本文字数： 1.7k 阅读时长 ≈ 2 分钟

物理层是计算机网络的最底层，主要负责数据的物理传输。

物理层为数据链路层提供服务，使数据链路层只需要考虑如何封装数据链路层协议数据单元（PDU），而不需要考虑数据是如何传输的。

注：有关计算机网络的相关概论知识，可参考以下：

  • 计算机网络概述

计算机网络基本重要概念

  • 因特网的组成：
    • 边缘部分： 由所有连接在因特网上的主机组成。
    • 核心部分： 由大量网络和连接这些网络的路由器组成。
  • 路由器： 连接两个或多个网络的设备，负责在多个网络之间转发数据包。
  • 交换方式：
    • 电路交换： 在数据传输之前，需要先建立一条专用的物理连接，数据传输过程中，这条连接一直存在。
    • 报文交换： 不需要建立连接，数据传输过程中，不需要建立连接。
    • 分组交换： 将数据分成多个小数据包，每个数据包独立传输，数据传输过程中，不需要建立连接。
  • 协议： 通信双方必须遵循的规则，包括：
    • 语法：数据与控制信息的结构或格式
    • 语义：需要发出何种控制信息，完成何种动作以及做出何种响应
    • 同步：事件实现顺序的详细说明
  • 服务： 在协议的控制下，两个对等实体间的通信使得本层能够向上一层提供服务，而与下层无关。
  • 接口： 上层使用下层服务的入口，下层为上层提供服务的出口。
  • 实体： 表示任何可发送或接收信息的硬件或软件进程。
  • 服务访问点： 在同一系统中相邻两层的实体交换信息的逻辑接口，称为服务访问点SAP。
    • 数据链路层： 数据链路层SAP称为帧的“头”和“尾”
    • 网络层： 网络层SAP称为IP地址
      • IP地址： 网络层地址，用于标识网络中的一个节点
    • 传输层： 传输层SAP称为端口
      • 端口： 传输层地址，用于标识一个进程
  • 服务原语：
    • PDU： 协议数据单元，对等实体之间传送的数据包称为协议数据单元。
    • SDU： 服务数据单元，层与层之间传送的数据包称为服务数据单元。
  • 连接唯一性：四元组（源IP地址，源端口，目的IP地址，目的端口）唯一标识一个连接

一、物理层的基本概念

1.1 物理层的功能

  • 定义物理接口特性
    • 机械特性： 接口的形状、尺寸、引脚数目和排列情况
    • 电气特性： 接口的电压范围、电流强度、阻抗匹配、传输速率等
    • 功能特性： 接口的信号线功能
    • 规程特性： 接口的规程
  • 定义传输介质的特性
  • 定义数据传输速率
  • 比特同步和时钟同步
  • 比特编码方式的定义

1.2 数据通信的基本概念

  • 数据（Data）：传送信息的实体
  • 信号（Signal）：数据的电气或电磁表现
  • 模拟信号：连续的信号
  • 数字信号：离散的信号

二、传输介质

2.1 导向传输介质

  1. 双绞线

    • 非屏蔽双绞线（UTP）
    • 屏蔽双绞线（STP）
    • 常用于以太网

  2. 同轴电缆

    • 基带同轴电缆
    • 宽带同轴电缆
    • 抗干扰能力强

  3. 光纤

    • 单模光纤
    • 多模光纤
    • 传输距离远，带宽大

2.2 非导向传输介质

  • 无线电波
  • 微波
  • 红外线
  • 激光

三、传输方式

3.1 串行传输与并行传输

  • 串行传输： 数据逐位按顺序依次传输
  • 并行传输： 数据同时多位按顺序依次传输

3.2 同步传输与异步传输

  • 同步传输： 数据块按固定长度依次传输
    • 外同步： 添加单独时钟信号线
    • 内同步： 数据中包含时钟信息
  • 异步传输： 数据块按不定长度依次传输

3.3 单工、半双工与全双工

  • 单工： 单向传输 无线电广播
  • 半双工： 双向交替传输 对讲机
  • 全双工： 双向同时传输 电话

四、信道与信道容量

4.1 信道的基本概念

  • 单工通信：单向传输
  • 半双工通信：双向交替传输
  • 全双工通信：双向同时传输

4.2 信道容量

  • 奈奎斯特定理：理想低通信道的最高码元速率
    • 公式：
    • ：带宽
    • ：电平数
  • 香农定理：信道的最大数据传输速率
    • 公式：
    • ：带宽
    • ：信噪比
  • 信噪比与带宽的关系

五、数字传输系统

5.1 编码与调制

  1. 基带传输

    • 不归零编码（NRZ）：存在同步问题
      • 正电平表示1，负电平表示0
    • 归零编码（RZ）：自同步，编码效率低
    • 曼彻斯特编码
      • 每个码元中间有跳变，负跳变表示1，正跳变表示0
    • 差分曼彻斯特编码

  2. 带通传输

    • 调幅（AM）
    • 调频（FM）
    • 调相（PM）

5.2 多路复用技术

  • 频分复用（FDM）
  • 时分复用（TDM）
  • 波分复用（WDM）
  • 码分复用（CDM）

六、物理层设备

6.1 中继器

  • 功能：放大信号
  • 特点：不能连接不同类型网络
  • 5-4-3规则

6.2 集线器

  • 功能：多端口中继器
  • 特点：共享带宽，冲突域
  • 工作方式：广播方式
